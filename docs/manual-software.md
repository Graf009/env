# Софт для ручной установки

Приложения, которых нет в Homebrew (cask) и в Mac App Store, поэтому
`brew bundle` их не ставит. Их нужно скачать и установить вручную — этот
файл держит список актуальным, чтобы на новой машине ничего не забыть.

Для каждого пункта: что это, почему вручную, откуда качать и как ставить.

## Hardware & Hobby

### CyberBrick Desktop App
**Приложение для программируемого конструктора CyberBrick (Bambu Lab).**
Модульная электроника + 3D-печать + визуальное/скриптовое программирование.
Десктопная версия даёт полный редактор конфигурации и связку с CAD и слайсером
Bambu Studio, с прямой загрузкой проектов в MakerWorld.

**Почему вручную:** нет Homebrew cask и нет в Mac App Store — распространяется
только как `.dmg` с сайта Bambu Lab.

**Требования:** macOS 10.11 (El Capitan) и выше.

**Установка:**
1. Открыть страницу загрузки: <https://bambulab.com/en-us/download/cyberbrick-apps>
2. Скачать `.dmg` для macOS (Desktop App).
3. Смонтировать и перетащить приложение в `Applications`.

**Связанное:** слайсер **Bambu Studio** ставится автоматически через
`Brewfile` (`cask "bambu-studio"`) — отдельно качать не нужно.

Документация: <https://wiki.bambulab.com/en/software/cyberbrick-apps/desktop>
